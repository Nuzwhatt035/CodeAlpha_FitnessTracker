Fitness Tracker App

A simple Android fitness tracking app that allows users to log and monitor daily fitness activities. Built as part of my internship project to practice Android development and data handling.

# Features
Add and track daily workouts
Log fitness data like time, exercise type, and calories
Simple and user-friendly UI
Lightweight and responsive design
# Tech Stack
Kotlin
Jetpack Compose 
Android Studio
Local storage / ViewModel 
room databse
# What I Learned
CRUD operations in Android
UI design and user input handling
Data management in mobile apps
Basic app architecture
